
# set-kata


## Build Status 

 [![Build Status](https://secure.travis-ci.org/dearfrankg/set-kata.png?branch=master)](http://travis-ci.org/dearfrankg/set-kata)

Simple object demo



## Getting Started
Install the module with: `npm install set-kata`

```coffeescript
Set = require('../src/set-kata.coffee')

set = new Set()
set.length()
set.add 55
set.doesContains 55
set.remove 55
```
